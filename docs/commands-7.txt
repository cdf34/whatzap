,list-pcs
    Get a list of current player characters.
,list-pcs-full
    Get a list of current player characters including descriptions.
,list-npcs
    Get a list of current NPCs.
,list-npcs-full
    Get a list of current NPCs including descriptions.
,commands $filter-text
    Get a list of commands. If $filter_text is set, gets all commands containing $filter_text as a substring.
,help
    Get a quick start help message.
